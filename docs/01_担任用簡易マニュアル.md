# 年間単元指導計画表 作成ツール ─ 担任用簡易マニュアル

> 本ツールはブラウザ上で動作します。インストール不要です。

---

## 画面の見方

| 領域 | 説明 |
|------|------|
| **ツールバー**（画面上部） | 学年選択・各種ボタンが並んでいます |
| **目標欄**（タイトル下） | 学校教育目標・学年目標・学級目標・担任情報を記入できます |
| **計画表**（メイン領域） | 縦に教科、横に月が並んだ表です |
| **セル** | 各教科×月の交差するマスに単元が表示されます |

---

## 基本操作

### 1. 学年を選ぶ

ツールバー左上のドロップダウンから学年（1年〜6年）を選択します。
学年ごとに対応する教科とカリキュラムデータが自動で切り替わります。

### 2. 目標欄を記入する

タイトル下の4つの欄をクリックすると編集できます。

- 学校教育目標
- 学年目標
- 学級目標
- 担任の専門・今年度の研究テーマ等

### 3. 単元を編集する

1. 編集したいセル（教科×月のマス）をクリック
2. 編集モーダルが開きます
3. 単元名・時数を入力
4. 「＋ 単元を追加」で同じセルに複数の単元を追加できます
5. **保存** ボタンで確定（キャンセルで元に戻ります）

> ☰マーク（ドラッグハンドル）をつかむと、単元の並び順を変更できます。

### 4. 教科をまとめて編集する

教科名（左端のセル）をクリックすると、**年間一覧編集**モーダルが開きます。

- 12か月分の単元を一覧で確認・編集できます
- 「← 1月 前にシフト」「1月 後にシフト →」で全単元を一括で月移動できます
- 単元を別の月にドラッグ＆ドロップで移動できます

### 5. 矢印（単元間のつながり）を描く

1. **矢印モード** ボタン（赤）をクリック
2. 始点となる単元をクリック
3. 終点となる単元をクリック → 矢印が描画されます
4. 続けて別の矢印を作成できます
5. **Escキー** または再度ボタンをクリックで終了

> 矢印を削除するには、矢印を **右クリック** または **ダブルクリック** します。

---

## 保存と読み込み

### JSON形式で保存する（推奨）

- **保存** ボタン → JSONファイルがダウンロードされます
- ファイル名例：`学級経営案_5年_2026-03-26.json`
- 単元データ・矢印・目標欄・設定がすべて保存されます

### JSON形式で読み込む

- **読み込み** ボタン → 保存したJSONファイルを選択
- すべてのデータが復元されます

### Excelで編集したい場合

1. **Excelテンプレート** ボタン → 現在のデータが入ったExcelがダウンロードされます
2. Excelで内容を編集します（`単元名(時数)` の書式を守ってください）
3. **Excelインポート** ボタン → 編集したExcelを読み込みます

---

## 印刷・PDF出力

1. **PDF出力** ボタンをクリック
2. ブラウザの印刷ダイアログが開きます
3. 用紙サイズ：**A3**、向き：**横** を選択
4. 「**PDFとして保存**」を選択してダウンロード

---

## 便利機能

| ボタン | 機能 |
|--------|------|
| **合計を表示** | 各教科の年間合計時数列を表示/非表示 |
| **抽出版** | 自己実現活動と関連する教科だけを抽出表示 |
| **リセット** | データを初期状態に戻す（確認あり） |
| **自己実現活動** ドロップダウン | 自己実現活動列の表示位置を変更 |
| **ヘルプ** | 各種マニュアルへのリンクを表示 |

---

## 困ったときは

| 症状 | 対処法 |
|------|--------|
| 編集内容が消えた | ページを閉じるとデータは失われます。こまめに **保存** してください |
| 矢印が描けない | **矢印モード**（赤ボタン）がオンになっているか確認してください |
| Excelインポートでエラーが出る | セルの書式が `単元名(時数)` になっているか確認してください |
| 印刷で色が出ない | 印刷設定で「背景のグラフィック」にチェックを入れてください |
| Excelテンプレートがダウンロードできない | インターネット接続を確認してください（Excel機能にはCDNライブラリが必要です） |
