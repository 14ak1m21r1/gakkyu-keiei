# 年間単元指導計画表 作成ツール ─ 詳細マニュアル

---

## 1. ツールの概要

本ツールは、小学校の年間単元指導計画表をブラウザ上で作成・編集・出力するためのWebアプリケーションです。

- **対応学年**: 1年〜6年
- **動作環境**: Google Chrome 推奨（他のモダンブラウザでも動作可能）
- **サーバー不要**: すべての処理がブラウザ内で完結します
- **データ保存**: JSON ファイルまたは Excel ファイルでローカルに保存

---

## 2. 画面構成

### 2.1 ツールバー

画面上部に固定表示されるコントロール領域です。

| 要素 | 説明 |
|------|------|
| **学年セレクタ** | 1年〜6年を切り替え。切り替え時にデータが初期化されます |
| **リセット** | 現在の学年のデータを初期カリキュラムに戻します |
| **矢印モード** | 単元間の関連を示す矢印の作成モードに切り替えます |
| **自己実現活動 位置** | 自己実現活動列の表示位置を変更（先頭/3番目/5番目/標準/最後尾） |
| **合計を表示** | 年間合計時数列の表示・非表示を切り替えます |
| **抽出版** | 自己実現活動に関連する教科・単元のみ抽出表示します |
| **保存** | 現在の計画をJSON形式でダウンロードします |
| **読み込み** | JSONファイルから計画を復元します |
| **Excelインポート** | Excelファイルから計画データを読み込みます |
| **Excelテンプレート** | 現在の計画をExcelファイルとしてダウンロードします |
| **PDF出力** | ブラウザの印刷機能でPDF出力します |
| **ヘルプ** | 各種マニュアルへのリンクを表示します |

### 2.2 タイトルバーと目標欄

タイトルバーの下に以下の4つの情報欄があります。各欄をクリックして自由にテキストを入力できます。

| 欄 | 記入内容 |
|----|----------|
| 学校教育目標 | 学校全体の教育目標 |
| 学年目標 | 該当学年の目標 |
| 学級目標 | 学級の目標 |
| 担任の専門・今年度の研究テーマ等 | 担任の専門性、校内研究テーマなど |

抽出版モード時はタイトル末尾に「【抽出版：自己実現活動】」が付加されます。

### 2.3 計画表（メインテーブル）

- **行**: 教科ごとに1行
- **列**: 12か月（4月〜3月）＋合計列
- **セル**: 教科×月の交差点。各セルに複数の単元を登録可能
- **色分け**: 学期ごとに背景色が異なります
  - 1学期（4月〜7月）: 薄い青
  - 8月: 薄い灰色
  - 2学期（9月〜12月）: 薄い黄色
  - 3学期（1月〜3月）: 薄い緑

### 2.4 教科一覧

学年により表示される教科が異なります。

| 学年 | 教科数 | 備考 |
|------|--------|------|
| 1〜2年 | 10 | 社会・理科・外国語・家庭なし |
| 3〜4年 | 13 | 家庭なし |
| 5〜6年 | 14 | 全教科 |

全教科一覧: 国語、算数、社会、理科、音楽、図画工作、体育、家庭、外国語、道徳・特活、健康、食育、自己実現活動、安全

## 3. 単元の編集

### 3.1 セル編集モーダル

データセル（教科×月のマス）をクリックすると編集モーダルが開きます。

**モーダルの構成:**

- **タイトル**: 「単元の編集」
- **サブタイトル**: 教科名と月名（例：「国語 ─ 4月」）
- **単元リスト**: 登録済みの単元が一覧表示されます

**各単元の入力要素:**

| 要素 | 説明 |
|------|------|
| ☰（ドラッグハンドル） | つかんでドラッグすると並び順を変更できます |
| 単元名テキスト入力 | 単元名を入力します |
| 時数入力 | 数値で時数を入力します（0以上） |
| ✕ボタン | その単元を削除します |

**操作ボタン:**

| ボタン | 動作 |
|--------|------|
| ＋ 単元を追加 | 新しい空の単元行を追加（時数の初期値は1） |
| キャンセル | 変更を破棄してモーダルを閉じます |
| 保存 | 変更を確定して反映します |

> モーダル外のオーバーレイ部分をクリック、またはEscキーでもキャンセルできます。

### 3.2 教科別一覧編集モーダル

教科名セル（左端）をクリックすると、その教科の12か月分を一覧で編集できるモーダルが開きます。

**主な機能:**

- 各月ごとにセクションが分かれ、単元の追加・編集・削除ができます
- 各月の合計時数がリアルタイムで表示されます

**月シフト機能:**

| ボタン | 動作 |
|--------|------|
| ← 1月 前にシフト | 全単元を1か月前に移動（4月のデータは消失） |
| 1月 後にシフト → | 全単元を1か月後に移動（3月のデータは消失） |

**ドラッグ＆ドロップ:**

- ☰ハンドルをつかんで、異なる月のセクションにドロップすると、単元を別の月に移動できます

### 3.3 メインテーブル上でのドラッグ

- 矢印モードがオフの場合、単元アイテムを同じ教科の別の月にドラッグ＆ドロップで移動できます
- 異なる教科間の移動はできません

---

## 4. 矢印機能（単元間の関連付け）

### 4.1 矢印の作成

1. **矢印モード** ボタン（赤）をクリックしてモードを有効化
2. 画面上部に黄色いバナーで案内が表示されます
3. 始点となる単元名をクリック（ハイライトされます）
4. 終点となる単元名をクリック → 矢印が描画されます
5. 続けて別の矢印を作成可能
6. Escキー、または矢印モードボタンを再度クリックで終了

**制約事項:**

- 同じ始点→終点の矢印は重複作成できません
- 矢印モード中はセルの編集はできません

### 4.2 矢印の削除

矢印の上で以下の操作を行います：

- **右クリック** → 確認ダイアログ → 削除
- **ダブルクリック** → 確認ダイアログ → 削除

### 4.3 矢印の表示

- 赤色の線で描画され、終点に三角形の矢じりが付きます
- 同じ教科内の矢印はS字カーブで描画
- 異なる教科間の矢印は教科間をまたぐカーブで描画
- ウィンドウサイズ変更時に自動的に再描画されます

## 5. データの保存と読み込み

### 5.1 JSON保存（推奨）

**保存** ボタンをクリックすると、以下の情報を含むJSONファイルがダウンロードされます。

| 項目 | 内容 |
|------|------|
| version | データ形式のバージョン（現在は1） |
| grade | 学年 |
| savedAt | 保存日時（ISO 8601形式） |
| jikoPos | 自己実現活動列の位置設定 |
| headerInfo | 目標欄の内容（学校教育目標・学年目標・学級目標・担任情報） |
| units | 全単元データ（教科×月ごとの単元名・時数） |
| arrows | 全矢印データ（始点・終点のID） |

ファイル名: `学級経営案_{学年}年_{日付}.json`

### 5.2 JSON読み込み

**読み込み** ボタンをクリックし、保存済みのJSONファイルを選択します。

- 確認ダイアログで承認後、現在のデータがすべて置換されます
- 学年・単元・矢印・目標欄・設定がすべて復元されます
- 不正なファイルの場合はエラーメッセージが表示されます

### 5.3 Excelテンプレートのダウンロード

**Excelテンプレート** ボタンをクリックすると、現在の計画データが入ったExcelファイルがダウンロードされます。

**Excelの構造:**

| 行 | 内容 |
|----|------|
| 1行目 | タイトル（学年名を含む） |
| 2行目 | ヘッダー（「月」＋教科名） |
| 3行目以降 | テキスト行（単元データ）と時数行（非表示）が交互に配置 |
| 最終行 | 年間合計（SUM数式） |

ファイル名: `{学年}年_年間単元指導計画表.xlsx`

### 5.4 Excelインポート

**Excelインポート** ボタンをクリックし、Excelファイル（.xlsx / .xls）を選択します。

**自動検出される情報:**

- 学年: タイトル行の「第○学年」から検出
- 教科: ヘッダー行の教科名から検出
- 月: A列の「○月」から検出

**セルの書式ルール:**

- 1つの単元: `単元名(時数)` 例: `銀色の裏地(5)`
- 複数の単元: 改行で区切り
- 時数がない場合: 単元名のみ記載（時数は0として扱われます）

> **注意**: Excelインポートでは矢印データ・目標欄は読み込まれません。アプリ上で再入力してください。

## 6. 印刷・PDF出力

### 6.1 操作手順

1. **PDF出力** ボタンをクリック
2. ブラウザの印刷ダイアログが表示されます
3. 以下の設定を推奨します:
   - **用紙サイズ**: A3
   - **向き**: 横
   - **余白**: 最小
   - **背景のグラフィック**: オン（色を印刷するために必要）
4. 「PDFとして保存」を選択

### 6.2 印刷される内容

- タイトルバーと目標欄
- 計画表の全体（学期の色分け含む）
- 矢印の接続線
- 合計列（表示中の場合）

### 6.3 印刷されない内容

- ツールバー
- 編集モーダル
- 矢印モードのバナー

> **ヒント**: 抽出版モードで印刷すると、自己実現活動に関連する教科だけのPDFが作成できます。

---

## 7. 抽出版モード

### 7.1 概要

**抽出版** ボタンをクリックすると、自己実現活動とそこに矢印で関連付けられた教科・単元のみが表示されます。

### 7.2 表示内容

- 自己実現活動列（オレンジ色でハイライト）
- 矢印で接続されている教科の列
- 接続されている単元のみ表示（その他の単元は非表示）
- 関連する矢印のみ表示

### 7.3 操作

- 抽出版モード中も矢印の追加・削除が可能です
- 「全体表示に戻す」ボタンで通常表示に戻ります
- データへの影響はありません（表示フィルターのみ）

---

## 8. データに関する注意事項

### 8.1 データの保持

- 本ツールにはサーバーがないため、**ページを閉じるとデータは失われます**
- 編集後は必ず **保存** ボタンでJSONファイルをダウンロードしてください
- 定期的な保存を推奨します

### 8.2 データの互換性

- JSON保存 → JSON読み込み: **完全互換**（単元・矢印・目標欄・設定すべて）
- Excelテンプレート → Excelインポート: **単元データのみ**（矢印・目標欄は未対応）
- 異なる学年のデータを読み込むと、学年が自動的に切り替わります

### 8.3 ブラウザの推奨設定

- JavaScript: 有効
- ポップアップ: ファイルダウンロードを許可
- インターネット接続: Excel機能の利用時に必要（SheetJSライブラリのCDN読み込み）
